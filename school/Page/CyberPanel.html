CyberPanel

OpenLiteSpeed + PHP5.3-7.0 一鍵安裝 / 可視化面板

簡介

CyberPanel，一款基於 OpenLiteSpeed 的 web 管理面板，支持自動添加 SSL、DNS、FTP、PHPMYAdmin 等。 下面是其環境一鍵安裝腳本，支持安裝 SSL、FTP、DNS、PHPMYAdmin，以及 PHP 版本支持 PHP5.3-PHP7.0 版本。

官網：http://docs.cyberpanel.net/doku.php?id=start

系統要求

Centos 7.x系統
Python 2.7
256MB以上內存
10GB以上空間

簡體版本一键安装

wget https://ppx.ink/CyberCP/install.tar.gz
tar zxf install.tar.gz
cd install
chmod +x install.py
bash install.shhttps://ppx.ink/cyberpanel.ppx

英文版一键安装

wget http://cyberpanel.net/install.tar.gz
tar zxf install.tar.gz
cd安裝
chmod + x install.py
python install.py [你的主機IP]

安裝完成後進入地址: https://IP:8199，使用用戶名: admin，密碼: 1234567 登錄即可。
